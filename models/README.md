# Models

This folder stores trained model artifacts generated by the training scripts (for example, .joblib files).

## Notes

- Model files are not tracked in git
- You can recreate everything by rerunning training:

  python -m src.train_audio
  python -m src.train_lyrics
  python -m src.train_fusion

## What you might see here

- audio_logreg.joblib
- audio_xgboost.joblib
- lyrics_logreg.joblib
- fusion_logreg.joblib
- label_encoder.joblib
